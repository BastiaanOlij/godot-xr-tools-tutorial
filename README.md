# Godot XR Tools demo

This is a Godot demo project in support of my [Godot XR Tools tutorial](https://youtube.com/playlist?list=PLe63S5Eft1KYUzXU_h4tCAyIeRCL90QgF) series.

In order to run this project you will Godot 3.5 or later.
